## New PC?

1. Using a terminal, run:
```sh
mkdir -p $HOME/.local/bin
```

2. [Install the `gh` cli]().

3. [Setup the `gh` cli as a credential helper](https://cli.github.com/manual/gh_auth_setup-git).

4. Auth to your AWS account.

5. Clone this repo.

6. Using a terminal, run:
```sh
cd /path/to/.dotfiles
make setup
```
